# Number Guessing Game

## Overview

The Number Guessing Game is a Java console-based application where the computer randomly generates a number, and the player must guess it within a limited number of attempts. The program provides hints after each guess, helping the player reach the correct answer.

This project demonstrates the use of Java fundamentals such as loops, conditional statements, user input handling, and random number generation.

## Features

- Random number generation
- User input using Scanner
- Hint system (Higher / Lower)
- Limited number of attempts
- Win/Lose messages
- Simple and interactive gameplay

## Rules of the Game

- If the guessed number is greater than the target number, the program asks the user to guess lower.
- If the guessed number is less than the target number, the program asks the user to guess higher.
- If the guessed number matches the target number, the player wins.
- If all attempts are exhausted, the game ends and the correct number is revealed.

## Technologies Used

- Java
- Scanner Class
- Math.random() Method

## Concepts Practiced

- Variables and Data Types
- Conditional Statements
- Loops
- User Input Handling
- Random Number Generation
- Basic Problem Solving

## How to Run

1. Compile the program:

```bash
javac NumberGuessing.java
```

2. Run the program:

```bash
java NumberGuessing
```
